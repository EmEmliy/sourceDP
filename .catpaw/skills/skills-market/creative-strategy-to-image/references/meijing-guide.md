# AI 广告创意设计 - 参考资料

## 常见广告类型 Prompt 模板

### 1. 促销活动类

```
[主体元素] + 热闹的促销氛围 + [颜色主题] + 现代简约风格 + 居中构图 + 高清质感
```

**示例**：
- "堆叠的礼品盒和红包，热闹的新年促销氛围，红色金色主题，现代简约风格，居中构图，高清质感"
- "大幅折扣标签和优惠券，限时促销氛围，橙黄色调，动感构图，视觉冲击力强"

### 2. 品质推荐类

```
[商品/食物] + 精致的摆盘/展示 + [风格描述] + 自然光线 + 侧光构图 + 摄影级画质
```

**示例**：
- "新鲜的水果拼盘和甜点，精致摆盘，日式小清新风格，自然柔光，侧光构图，摄影级画质"
- "高端电子产品，简洁的产品展示，商务极简风格，影棚灯光，正面构图，产品级精修"

### 3. 场景故事类

```
[人物场景] + [情绪表达] + [场景氛围] + [风格] + 广角构图 + 氛围感强
```

**示例**：
- "年轻人在餐厅用餐的快乐场景，朋友聚会氛围，现代生活方式风格，广角构图，温馨氛围感强"
- "一家人在家中共度时光，温馨的家庭氛围，暖色调家庭风格，中景构图，充满幸福感"

### 4. 社交媒体类

```
[内容主题] + 朋友圈风格 + [色调] + [构图] + 质感强 + 适合手机浏览
```

**示例**：
- "美食打卡分享，朋友圈风格，暖色调美食图，特写构图，质感强，适合手机竖屏浏览"
- "旅行打卡照，朋友圈风格，清新蓝绿调，广角构图，氛围感强，适合手机竖屏浏览"

## 尺寸与构图对应表

| 比例 | 场景 | 构图建议 |
|------|------|---------|
| 16:9 | 朋友圈、banner、PPT | 左右分割、居中对齐、宽幅风景 |
| 9:16 | 信息流、抖音、小红书 | 上下分割、中心聚焦、竖版人物 |
| 1:1 | 电商主图、头像、ins | 居中构图、方形商品展示 |
| 4:3 | 小红书、微博 | 平衡构图、故事感 |
| 3:4 | 微博、空间 | 竖向叙事、人物为主 |

## 颜色情绪对应表

| 颜色 | 情绪/用途 | 适用场景 |
|------|----------|---------|
| 红/橙/黄 | 促销、热情、紧迫感 | 促销活动、限时优惠 |
| 蓝/青 | 专业、信任、科技 | B端产品、科技服务 |
| 绿 | 健康、自然、安全 | 食品、农产品、健康 |
| 粉/紫 | 女性、温柔、浪漫 | 美妆、母婴、礼品 |
| 黑/白/灰 | 简约、高端、品质 | 高端产品、奢侈品 |
| 棕/金 | 品质、传统、尊贵 | 茶叶、酒类、礼品 |

## 常见失败原因与解决

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 生成内容与预期不符 | Prompt 不够具体 | 增加主体、风格、构图描述 |
| 文字乱码 | 模型中文渲染能力弱 | 换 LongCat-Image |
| 人物变形 | 复杂姿态超出能力 | 换即梦 4.5（人物一致性） |
| 颜色不对 | 风格描述不明确 | 明确色调关键词 |
| 构图奇怪 | 构图词不准确 | 使用标准构图词（居中/对称/三分） |
