# 创意策划方法论

## 一、核心三角公式

```
创造需求 → 心智种草 → 效果转化
```

| 阶段 | 用户心理 | 做功点 |
|------|----------|--------|
| **开头（创造需求）** | 避坑/理想身份/择优/落后/与我有关/悬念好奇 | 黄金3秒（吸睛） |
| **中间（心智种草）** | 产品属性/利益点/价值观 | 7秒决策（转化） |
| **结尾（效果转化）** | 呼吁引导/自然不生硬/减弱防备心 | 强引导 |

## 二、速成爆文4维公式

| 维度 | A | B | C | D |
|------|---|---|---|---|
| **开头** | 场景化营销 | 故事化脚本 | 低价策略 | 微剧情埋梗 |
| **中间** | 卖点展示/解决痛点+解决信任 |
| **结尾** | 强化引导+售后无忧+提示转化 |

### 3大视角
- **商家视角**：农销、厂家、店家
- **用户视角**：开箱、种草、第三人称
- **官方视角**：官方发布、官方福利

## 三、开头类型与话术模板

### A. 场景化营销
| 类型 | 说明 | 话术模板 |
|------|------|----------|
| 特定区域 | 地域稀缺感 | "这是我们新疆特有的XXX" |
| 工厂仓库/产地 | 原料展示建立信任 | "xx大量到货，为了缓解仓库压力，拿出一批做福利" |
| 家用场景 | 生活代入感 | "早上来不及吃早饭的家人们一定要入手xxx" |
| 吃播直拍 | 视听结合降低广告感 | 吃播画面+信息标题 |
| 及时速达 | 强调配送速度 | "今天下单今天就到"/"最快30分钟送达" |

### B. 故事化脚本
| 类型 | 说明 | 话术模板 |
|------|------|----------|
| 圈定人群 | 用爆品话题吸引留住用户 | "家里有老人小孩的一定要买点XXX" |
| 设定情景 | 设置痛点带入感 | "这么热的天，谁还出去买XXX啊" |
| 话题科普 | 朗朗上口贴近生活 | "秋吃螃蟹，冬吃蚝，现在XX正是肥美的时候" |

### C. 低价策略
| 类型 | 说明 | 话术模板 |
|------|------|----------|
| 饥饿营销 | 限时稀缺感 | "XXX是我们今年最后一批了，卖完就没有了" |
| 低价诱惑 | 薅羊毛心理 | "这波羊毛你爱薅不薅"/"个位数还包邮" |
| 重复手法 | 重复强调 | "个位数抢五斤，个位数抢五斤"/"好消息好消息" |

### D. 微剧情埋梗
| 类型 | 说明 | 话术模板 |
|------|------|----------|
| 矛盾冲突 | 夸张冲突吸引停留 | 上门闹事："是不是你说的，xx只要1分钱起...." |
| 对话式 | 降低广告感 | A:"老板你这xx也太大了" B:"咱这是XX产地" |
| 提问式 | 引发好奇 | A:"你猜猜我这XX多少钱？" B:"20块？" |
| 吸睛猎奇 | 反常识 | A:"大哥，你家橘子咋卖" B:"这不是橘子，这是沃柑" |

## 四、中间段：卖点展示（五感法）

| 感官 | 话术示例 | 时长 |
|------|----------|------|
| 视觉 | "皮薄肉厚，果肉金黄饱满"/"泡沫绵密" | 1-2s |
| 听觉 | "咬一口，emmm~~"（咀嚼、嗦粉吸溜声效） | 1-3s |
| 嗅觉 | "刚拿到手奶香味就扑鼻而来" | 1-2s |
| 触觉 | "汁水超多，一挤就爆汁"/"软乎乎的" | 1-2s |
| 味觉 | "香甜软糯，像冰淇淋一样丝滑" | 1-2s |

### 解决信任
| 类型 | 话术模板 |
|------|----------|
| 官方活动 | "美团年终活动"/"美团送福利" |
| 农民自营 | "俺们农民刚学会线上卖，给大家源头地头价" |
| 价值肯定 | "家里老人小孩都爱吃啦"/"不行，吃完我得再去囤几箱" |
| 参与门槛 | "不用拼单；不用砍价；不用拉人；没有套路" |

## 五、结尾段：效果转化

| 类型 | 做功点 | 话术模板 |
|------|--------|----------|
| 强化引导 | 利益点+稀缺感+紧迫感 | "关键现在只要个位数，限时2000件" |
| 售后无忧 | 品质保障打消顾虑 | "日期新鲜"/"有坏的我全赔" |
| 提示转化 | 明确点击路径 | "点击视频下方链接，搜索xxx就能购买了" |

## 七、制作加分准则

### BGM策略
- **营造惯性**：使用实时热门BGM，让用户惯性停留
- **营造期待**：特定BGM当"标签"，如李佳琪=商品便宜
- **营造情绪**：选择合适声音预设情绪

### 演员要素
- **跑量女演员**：面部圆润饱满/笑容亲切朴实/不浓妆/不网红脸
- **跑量男演员**：沉稳自然/干净阳光/不可带攻击性/不油腻/下沉原生
- **核心公式**：真实感=代入感；信赖感=专业感；亲切感=观众缘

### 剪辑节奏
- 前5s至少2个以上镜头切换
- 30s视频至少10+个镜头，2-4s剪一镜
- 避免长时长静止镜头，可通过拉大、缩放让镜头运动
- 时长建议：纯剪辑20s左右，口播30s内，剧情50s内

### 加分项
- **场景爆**：奢华精美/豪车别墅/下沉农村工地
- **关系爆**：前男友女友/老板/父母子女/同事攀比
- **演员爆**：表演浮夸or原生/形象吸睛/要么潮要么土
- **音效爆**：抖音top火爆音乐/环境音效（油炸等）

## 八、低成本迭代策略

### 视频迭代
| 策略 | 品类 | 内容 | 说明 |
|------|------|------|------|
| 跑量复刻 | 一致 | 一致 | 同品同内容，文案套路和剪辑优化迭代 |
| 素材衍生 | 一致 | 一致 | 镜像反转、更换镜头等产出新素材 |
| 品类衍生 | 不同 | 一致 | 同品类不同商品，文案套路一致 |
| 文案衍生 | 不同 | 不同 | 跑量文案套路更换其他品类 |

### 图片迭代
| 策略 | 品类 | 呈现方式 | 说明 |
|------|------|----------|------|
| 跑量复刻 | 相似 | 一致 | 相似商品，相同摆放 |
| 相似衍生 | 一致 | 相似 | 同商品，相似摆放 |
| 素材衍生 | 一致 | 不一致 | 同商品，不同摆放 |
| 品类衍生 | 同品类 | 一致 | 不同商品，相同摆放 |

## 九、爆品选品公式

```
爆品因素 = 需求 × 决策成本 × 内容性
```

- **需求**：人群偏好、大众vs小众、实用性vs颜值度、季节性vs长期刚需
- **决策成本**：体感价格VS认知价格差异、竞品价格优势
- **内容性**：媒体热度、认知度、品牌vs白牌vs网红品、可视化功能点
