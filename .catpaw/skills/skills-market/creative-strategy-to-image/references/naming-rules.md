# 素材命名规范

## 代理商产出命名

```
{业务}-{活动}-{会场名}-{周期}-代产-{代理商名称}-{地区}-{大品类}-{具体选品}-{价格}-{尺寸}-{时间}-{制作人}-{条数}
```

**示例**：
```
闪购大促-双旦大促-主会场-圣诞-代产-洪星-全国品-酒类-劲酒-13.9元-720竖-20241225-张三-01
```

## 内部产出命名

### 白盒（半自动化）
```
{业务}-{活动}-{会场名}-{周期}-自动化-白盒-{模板名称}-{大品类}-{具体选品}-{价格}-{尺寸}-{时间}-{制作人}-{条数}
```

### 黑盒（全自动化）
```
{业务}-{活动}-自动化-黑盒全自动-{模板名称}-{大品类}-{具体选品}-{价格}-{尺寸}-{时间}-{制作人}-{条数}
```

## 字段说明

| 字段 | 可选值 | 说明 |
|------|--------|------|
| 业务 | 闪购大促 | 业务线标识 |
| 活动 | 双旦大促/618/双11等 | 活动名称 |
| 会场名 | 主会场/分会场 | 投放会场 |
| 周期 | 圣诞/元旦/全周期 | 投放时间周期 |
| 来源 | 代产/自动化 | 素材来源 |
| 地区 | 全国品/地区品 | 投放地区 |
| 价格 | xx元/满xx减xx元 | **必须带"元"字** |
| 尺寸 | 720竖/1280横 | 素材尺寸 |

## 特殊标签

| 场景 | 额外标签 |
|------|----------|
| 开屏专属 | 【开屏专属】 |
| 代投素材 | 「代投」 |
| 自投素材 | 「自投」 |

## 实验测试类素材命名

当素材用于AB测试/人群实验时，命名需包含**人群标识**字段，方便数据回收归因。

### 命名规则
```
{业务}-{人群}-{场景}-{品牌}-{素材类型}-{版本号}
```

### 关键原则
- **人群字段用中文**：如"白领""尝鲜""学生""家庭"，搜索即可定位该人群全部素材
- 无需英文标签，中文字段本身具有唯一性
- 版本号用于同一组合下的多版本迭代（v01/v02/v03）

### 示例（到餐新客场景测试）
```
到餐新客-白领-工作餐-肯德基-场景图-v01
到餐新客-白领-周末聚餐-烤肉-大字报-v01
到餐新客-尝鲜-出片打卡-鲍师傅-种草图-v01
到餐新客-尝鲜-种草爆款-山缓缓-场景图-v01
```

### 数据回收
搜索"白领"或"尝鲜"即可分别定位两波素材，按场景/品牌/素材类型进一步拆分分析。
