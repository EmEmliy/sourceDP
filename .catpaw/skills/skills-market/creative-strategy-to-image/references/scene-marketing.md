# 闪购场景化营销策略

## 背景
- 爆品已趋向固化，长期反复使用相同爆品难以有效提升转化率
- 场景化素材可Cross其他非闪购爆品与闪购场景结合，提升吸引力
- 可将非外投爆品转变为跑量爆品，扩充爆品池

## 16大场景分类

### S级场景（头部）
| 场景 | 用户需求 | 适配爆品 | 话术模板 |
|------|----------|----------|----------|
| **夜间采购** | 晚上出门不便、临时需求、夜间嘴馋/加班 | 零食饮料、洗护用品、啤酒 | "烧烤配酒，越喝越有，大晚上撸串怎么能没有啤酒" |
| **买菜做饭** | 日常采购&应急采购，临时缺食材 | 瓜果鲜蔬、火锅类、厨具 | "节俭的阿婆最近居然吃起了xx，1分钱起" |
| **重物到家** | 独居女性采购重物费时费力 | 大桶水、西瓜、行李箱、洗衣液 | "现在谁还买东西自己拎啊" |
| **下午茶** | 水果尝鲜、下午补充能量、同事分享 | 果切、蛋糕、饮料 | "在下午来上一杯甜甜的奶茶+1分钱起蛋糕" |
| **天气应急** | 突发雨雪、变冷变暖 | 雨伞、雨披、夏凉被、棉被 | "本来好好的天气说变就变，还好1分钱起买了伞" |

### A级场景（腰部）
| 场景 | 用户需求 | 适配爆品 |
|------|----------|----------|
| **日间采购** | 懒宅不愿出门（天气热/刷剧打游戏） | 酒水饮料、零食、日用 |
| **采购囤货** | 节省时间、一次性储备大批量 | 酒水饮料、零食、瓜果 |
| **身体不适** | 生病时紧急订购清淡食物/药品 | 退热贴、体温计、黄桃罐头 |
| **宠物需求** | 为宠物购买食物、护理用品 | 猫粮、狗粮、猫罐头 |

### B级场景（闲置/特定时期）
| 场景 | 适用时机 | 适配爆品 |
|------|----------|----------|
| **家庭陪伴** | 亲子群体 | 尿不湿、婴儿湿巾、奶粉 |
| **职场共餐/居家聚会** | 聚餐场景 | 酒水饮料 |
| **情侣约会** | 约会/送礼 | 鲜花、美妆、酒水 |
| **出行应急** | 本地玩乐/异地出行/事件应急 | 防晒、数据线、计生用品 |

## 节日场景
| 节日 | 核心品类 | 话术方向 |
|------|----------|----------|
| **元旦/春节** | 坚果、白酒、红包、砂糖橘 | "眼瞅过年了，家里怎能不备上xx" |
| **情人节/520/七夕** | 玫瑰花束、巧克力、护肤品 | 情感礼赠 |
| **妇女节/母亲节** | 康乃馨、面膜、香水 | 关爱女性 |
| **清明节** | 青团、春茶 | 时令节气 |
| **端午节** | 粽子、咸鸭蛋、艾草 | 传统民俗 |
| **中秋节** | 月饼礼盒、柚子、葡萄 | "月是故乡明，心安是归途" |
| **节气时令** | 开海季大闸蟹、梭子蟹 | 时令尝鲜 |

## 热点场景
- 实时热点：媒体热点、年轻向热梗
- 案例：过年"炫砂糖橘"、24年第一口梭子蟹、干噎酸奶

## 场景素材要点
- 前5秒：一目了然交代具象使用场景
- 随后展现：利益点 + 配送服务
- 场景定义包含：WHO（购买者/收件人）、WHEN（时间）、WHERE（位置）、WHAT（商品）、WHY（动机）
