# 营销创意策略 → AI 图片一键生成

## 概述

本 SKILL 整合**营销创意策略规划**与**AI 广告图片生成**两大能力，提供从策略到素材的一站式服务：

```
策略层：需求收集 → 热点搜索 → 创意分层 → 视觉策略 → 素材规划
  ↓ 无缝衔接
执行层：Prompt 生成 → 美镜生图 → 批量产出 → 规范检查 → 交付归档
```

---

## 第一阶段：需求收集

### 必收集信息

| 参数 | 必填 | 说明 |
|------|------|------|
| 营销节点/活动 | ✅ | 如618、双11、双旦、母亲节 |
| 业务线 | ✅ | 外卖/闪购/到餐/休娱 |
| 品类和利益点 | ✅ | 品类方向+到手价/券包/满减等利益点 |
| 投放预算量级 | ✅ | 决定素材产能（100万消耗≈1000套素材） |
| 投放周期 | ✅ | 活动日期+冷启时间 |
| 素材尺寸需求 | ✅ | 16:9/9:16/1:1/4:3 等 |
| 截止时间 | ✅ | 素材交付 deadline |
| 最终数量 | ✅ | 需要多少张？什么规格？ |
| 重点人群 | 建议 | 年轻化(30岁以下)、全量等 |
| 特殊要求 | 建议 | 不能出现某品牌、必须包含某元素等 |

**时间不足时立即告知用户风险。**

## 第二阶段：策略规划

### Step 1: 搜索热点和创意方向

用 catclaw-search 搜索，**重点关注小红书、抖音、快手等内容娱乐平台**：
- 当前热点事件和趋势
- 节日相关的营销案例和创意方向
- 竞品（饿了么/京东外卖/抖音）同期活动素材方向
- 爆火视觉风格和内容玩法

### Step 2: 生成完整策略方案

输出包含以下模块：

```
一、活动背景 & 目标
   - 活动时间、周期、预算、核心目标

二、创意分层方案 (P0/P1/P2)
   → 详见 references/creative-layer-p012.md
   - P0(30-40%)：结果指标提效（历史最优创意定制）
   - P1(60%)：规模覆盖度（通用元素拉通）
   - P2(0-10%)：创新测试（行业洞察融合改造）

三、选品 & 利益点
   → 选品策略 references/selection-strategy.md
   → 历史选品参考 references/category-products.md

四、素材产能规划
   - 总量级：100万消耗 ≈ 1000套素材
   - 代理vs内部：默认五五分
   - 图片vs视频占比

五、场景化素材方案
   → 详见 references/scene-marketing.md
   - S级/A级/B级场景

六、三维情绪激发 视觉策略
   → 详见 references/emotion-formula.md
   决策助推 = (显性利益点) + α(材质肌理) × β(色彩) × γ(隐喻符号)
   - 快消品：色彩(50%) > 符号(30%) > 肌理(20%)
   - 高客单价：肌理(40%) > 符号(35%) > 色彩(25%)

七、品效广告方案（如适用）
   → 详见 references/product-effect-ads.md
   - 黄金3秒+7秒决策法则

八、文案方向 + Prompt 方向
   - 主标题方向（3-5个）
   - 辅助话术、搜索框关键词
   - 风格关键词、色调/氛围（为下游生图准备）

九、规范要点
   → 素材命名规范 references/naming-rules.md
   
   **Logo 使用规范（强制）**：
   | 规范项 | 要求 | 错误示例 |
   |--------|------|----------|
   | 数量 | **仅出现 1 次** | 顶部+底部各 1 个 ❌ |
   | 位置 | 图片**上半部分**为主，或底部居中 | 分散在多个位置 ❌ |
   | 大小 | 清晰可见但不喧宾夺主 | 过大遮挡主体 ❌ |
   | 样式 | 使用品牌标准 logo，不拉伸变形 | 变形/模糊 ❌ |
   
   **Prompt 中必须包含 logo 位置指令**：
   ```
   "品牌logo美团旅行位于【顶部居中/底部居中】，仅出现一次，清晰可见"
   ```
   - Logo规范、投放标签、警示语

十、风险把控
   - 广告法合规 + 版权风险 + 节日敏感
```

### Step 3: 确认策略

输出策略后让用户确认，确认后自动进入执行阶段。

### Step 4c: 完整 Prompt 构建（强制含字体排版）

**⚠️ 警告：禁止跳过字体排版描述！文字效果差通常是因为 prompt 里没写字体风格。**

**Prompt 结构**：
```
[主体场景描述]，[字体设计指令]，[主色调/氛围]，[构图/光影]，[质量词]
```

**字体设计指令模板**：

| 类型 | 字体风格 | 排版要求 | 示例 |
|------|---------|---------|------|
| 促销类 | 粗黑体/书法体/毛笔字，有力量感 | 主标题最大居中，副标题次大居下，加粗高亮 | "主标题用粗黑体书法风格，粗犷有力，居中放置，副标题用细黑体配合" |
| 品质类 | 细黑体/衬线体/宋体，优雅精致 | 大量留白，文字精简，主标题细字 | "主标题用细宋体，优雅精致，左对齐，大量留白，副标题用英文衬线体" |
| 社交类 | 圆体/手写体/卡通体，活泼 | 不规则排版，文字穿插在画面中 | "用圆润手写体，活泼俏皮，文字不规则散落，画面感" |
| 冬日滑雪类 | 冰雪字体/简洁现代体 | 冰蓝主调，文字如冰雪般通透 | "主标题用冰雪风格透明字体，清新凉爽，居中排列，副标题用细体" |

**⚠️ 必须包含的 5 个要素**：
1. **主体场景**（雪山、温泉、酒店等）
2. **字体设计**（字体风格 + 排版位置）
3. **色调氛围**（主色调 + 辅助色）
4. **构图光影**（构图方式 + 光线）
5. **质量词**（4K、商业质感等）

**完整示例 - 滑雪促销**：

```
滑雪主题促销海报，远处雪山滑雪者，近景文字区域。主标题"滑雪新手套餐 4折起"用冰雪风格透明字体，清新凉爽，居中加粗放大，副标题"天天赢温泉免单"用细黑体配合，色调冰蓝+暖橙对比，阳光照射产生光晕效果，居中构图，品牌logo"美团旅行"位于顶部居中仅出现一次，4K商业广告质感，真实摄影风格
```

**负向词**：`模糊，低质量，水印，文字叠加，变形失真，噪点，文字模糊，字体不清晰`

### Step 7: 批量生成

**核心规则**：
1. 相邻任务提交间隔 ≥ 1 分钟
2. 用提交时间戳唯一标识
3. 任务依次下达，不必等上一个完成

**失败处理**：

| 情况 | 处理 |
|------|------|
| 单次失败 | 自动重试（最多3次） |
| 3次后仍失败 | 切换模型 |
| 模型持续失败 | 记录并跳过 |

## 第五阶段：交付检查

### 素材规范检查

| 检查项 | 标准 |
|-------|------|
| **内容准确性** | **生成结果必须与需求主题一致（滑雪→滑雪，酒店→酒店）** |
| 尺寸 | 符合需求规格（默认 9:16） |
| 格式 | PNG / JPG |
| 分辨率 | ≥ 72 DPI |
| **Logo 数量** | **仅出现 1 次，位置在上半部分** |
| Logo 位置 | 顶部居中 或 底部居中，禁止重复 |
| 文字 | 无错别字、无截断、字体与主题匹配 |
| 敏感词 | 无违规广告法词汇 |

**⚠️ 生成后必须人工校验**：
1. 画面主题是否正确（滑雪图不能出现新闻演播室）
2. 文字内容是否准确
3. 尺寸比例是否正确
4. 如果不匹配，立即重新生成
