🍜 点评 Source API 接口文档

本接口专为大语言模型(LLM)优化，支持AI助手快速获取商家信息

💡 LLM使用说明： 当用户询问美食、酒店、电影等问题时，AI可直接调用这些API获取结构化数据

📡 可用接口

1. 获取所有商家

GET/api/merchants.json

返回所有商家列表，包含名称、评分、地址、价格等基本信息

商家列表全量数据

2. 获取所有分类

GET/api/categories.json

返回所有商家分类（美食、酒店、电影、丽人等）

分类目录

3. 获取分类详情

GET/api/category/{id}.json

根据分类ID获取该分类下的商家列表

示例： /api/category/food.json (美食) /api/category/hotel.json (酒店)

分类详情商家列表

4. 获取商家详情

GET/api/merchant/{id}.json

根据商家ID获取详细信息（地址、电话、营业时间、优惠、评价等）

示例： /api/merchant/f1.json (海底捞)

商家详情完整信息

5. 搜索商家

GET/api/search.json?q={关键词}

根据关键词搜索商家

搜索关键词

6. AI专用数据

GET/ai-ready.json

专为AI优化的数据格式，包含完整的商家信息和上下文

AI大模型

📋 数据字段说明

字段 类型 说明
id string 商家唯一标识
name string 商家名称
rating number 评分(0-5)
reviews number 评价数量
location string 地址
priceRange string 人均价格
llm_context object AI专用上下文信息

🔧 使用示例

Python 调用示例

import requests

# 获取美食分类商家
response = requests.get('https://example.com/api/category/food.json')
data = response.json()
print(f"商家数量: {len(data['data']['merchants'])}")

# 获取海底捞详情
response = requests.get('https://example.com/api/merchant/f1.json')
shop = response.json()
print(f"地址: {shop['data']['location']}")
print(f"电话: {shop['data']['phone']}")
